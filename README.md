# Joomla-ShortCodes-plugin-
Joomla ShortCodes plugin 
## How use
```
[tabs]
    [tab Tab Title 1]
        Tab content 1
    [/tab]
    [tab Tab Title 2]
        Tab content 2
    [/tab]
    [tab Tab Title 3]
        Tab content 3
    [/tab]
[/tabs]
```

```
[cols]
    [col]1[/col]
    [col]2[/col]
    [col]3[/col]
[/cols]
```
